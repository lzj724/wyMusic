# wymusic

> 仿网易云音乐手机版功能，除登陆才能使用的功能以外，都已经实现了，使用的框架为Vue
> 

##技术栈
>使用的以vue全家桶为基础，技术层面比较简单。
>数据是使用的[Binaryify](https://github.com/Binaryify/NeteaseCloudMusicApi)提供的网易api接口，在此感谢！
> Vue
> Vuex
> Router
> Sass
>
